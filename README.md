# **Consumer Behavior Analysis – Retail Analytics Project**

## **Overview**

This project analyzes a retail company’s customer shopping behavior to uncover what drives purchases, loyalty, and engagement. Using a combination of Python, SQL, and Power BI, the project identifies key trends across demographics, product categories, and sales channels. The goal is to help the business improve sales, enhance customer experience, and strengthen long-term loyalty.

---

## **Business Question**

**How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing and product strategies?**

## **Deliverables**

### **1. Data Preparation & Modeling (Python)**

* Cleaned missing values, duplicates, and inconsistent entries
* Engineered features for customer segmentation and purchase behavior
* Prepared data for SQL analysis and BI reporting

### **2. Data Analysis (SQL)**

* Structured the dataset into business-ready tables
* Simulated transactions and customer activity
* Queried key metrics such as:

  * High-value customer segments
  * Repeat purchase patterns
  * Impact of discounts, reviews, seasons, and payment methods

### **3. Visualization & Insights (Power BI)**

* Built an interactive dashboard showing:

  * Sales trends
  * Category performance
  * Customer loyalty metrics
  * Online vs. offline behavior
* Designed for stakeholder decision-making

### **4. Report & Presentation**

* Summarized analytical approach, key findings, and business recommendations
* Highlighted actionable insights to guide strategy in marketing, product planning, and customer engagement

### **5. GitHub Repository**

This repository includes all scripts, SQL queries, and dashboard files needed to reproduce the analysis:

* Python scripts for data cleaning and feature engineering
* SQL queries for segmentation and trend discovery
* Power BI files for visualization
* Final report and presentation
